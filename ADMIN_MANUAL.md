# Мануал по админ-панели Regola

## Вход

Админка открывается по адресу из переменной `VITE_ADMIN_PATH`.
Если переменная не задана, путь по умолчанию:

```text
/_secure-admin-7f29A228lswP
```

Для входа нужны три значения из `.env`:

```env
ADMIN_LOGIN=your_login
ADMIN_PASSWORD=your_strong_password
ADMIN_ACCESS_KEY=your_secret_key
```

После изменения `.env` перезапустите сервер.

## Добавить товар

1. Откройте админ-панель.
2. Заполните название, описание, остаток.
3. Укажите основную цену или цены маркетплейсов.
4. Добавьте ссылки Wildberries, Ozon, Яндекс Маркета, если товар там продаётся.
5. Добавьте изображение:
   - вставьте URL картинки;
   - или выберите файл с компьютера.
6. Нажмите `Добавить товар`.

## Цены

Цены ставятся вручную.

- `Основная цена` — запасная цена товара.
- `Цена Wildberries`, `Цена Ozon`, `Цена Яндекс Маркета` — показываются в карточке товара отдельными строками.
- Если цена площадки не нужна, оставьте поле пустым.

Автоматической загрузки цен с маркетплейсов нет.

## Редактировать товар

1. Найдите товар в списке ниже формы.
2. Нажмите `Редактировать`.
3. Измените поля.
4. Нажмите `Сохранить`.

## Удалить товар

1. Найдите товар в списке.
2. Нажмите `Удалить`.
3. Товар сразу пропадёт из каталога.

## Если не пускает в админку

1. Проверьте `ADMIN_LOGIN`, `ADMIN_PASSWORD`, `ADMIN_ACCESS_KEY` в `.env`.
2. Перезапустите сервер.
3. Очистите localStorage браузера для домена сайта или нажмите выход, если сессия устарела.
4. Убедитесь, что пароль не короче 12 символов.
